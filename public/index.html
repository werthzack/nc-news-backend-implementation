Article API Documentation

A simple guide to using the Article API.

  • GET /api/topics
  • GET /api/articles/:article_id
  • GET /api/articles
  • GET /api/articles/:article_id/comments
  • POST /api/articles/:article_id/comments
  • PATCH /api/articles/:article_id
  • DELETE /api/comments/:comment_id
  • GET /api/users
  • GET /api/articles (queries)
  • GET /api/articles/:article_id (comment count)

GET /api/topics

Retrieves a list of all topics in the database.

Example Request

GET http://localhost:4090/api/topics

Example Response

{
"topics": [
    { "id": 1, "slug": "Sports", "description": "Covers all sports from football to baseball" },
    { "id": 2, "slug": "Cooking", "description": "Hey good looking, what you got cooking?" }
]
}
